# Design History

Archived HTML sign-off mockups from earlier design rounds. **Not live code.**
Kept for reference only — the live app is under `app/` and does not reference
anything in this directory.

## Contents

- `prototype/` — original prototype pages (agent hub, community pill, upload
  flows, tiktok grid, landing v3, etc). Formerly served under `/prototype`.
- `prototypes/` — misc later prototypes (formerly served under `/prototypes`).
- `design-mocks/` — six sign-off design directions (editorial mono, pixieset
  gallery, compass luxe, cinematic runway, aman hotel, naftali light) plus
  their swipe variants. Formerly served under `/design-mocks`.

Moved out of `public/` on 2026-07-11 as part of the post-rebrand cleanup so
these static HTML files stop being publicly served and stop showing up in
production route trees.
