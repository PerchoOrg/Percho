Percho AgentHub · my-listing tabs redesign

5 tabs redesign — 4 prototypes to compare

Today the 5 sub-tabs (Details · Media · Marketing · Leads · Analytics) live in a horizontal pill row that overflows on mobile (only ~3 fit). Each variant below shows the same Desktop ↔ Mobile pair so you can pick the direction. A/B/C explore vertical layouts; D stays horizontal but borrows Amazon Grocery's icon-chip pattern.

Quick read

Going fully vertical on a 375 px phone (A) steals horizontal real estate from the panel content. B compromises with an icon-only mobile rail but loses labels. C keeps horizontal on desktop and switches to a vertical stacked card list on mobile — every tab fully labelled, no overflow. D stays horizontal everywhere using Amazon-style circular icon chips — visually distinct from any text-tab nav above/below, but still requires mobile scroll. C and D are the strongest candidates — pick C if "all 5 visible no scroll" matters most, pick D if visual identity / consistency with desktop matters most.
