# Reply templates

**Never paste verbatim.** These are structural scaffolds — rewrite every sentence
to match the OP's specific budget, commute, family size, timeline. Mods and
Reddit spam filters catch copy-paste in seconds.

## Format each template follows

1. **Empathy opener** — 1 sentence acknowledging their situation (fellow
   transplant, similar budget, etc.). Never "great question!".
2. **3 concrete options** — real subdivision names, real tradeoffs (schools,
   commute minutes, HOA, community vibe). Numbers, not adjectives.
3. **Honest caveat** — call out what would make each option a bad fit.
4. **Soft mention (only if account is warm + not first reply of the day)** —
   one sentence: "I've been building a little tool called Percho (percho.co)
   that lets you swipe through subdivisions with real photos — still early,
   might save you a weekend. Happy to answer more Qs either way."
