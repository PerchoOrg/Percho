# Percho Master Music Library — v1

Royalty-free instrumental tracks vetted for Percho's brand: modern, premium, warm, PNW-lifestyle. No cinematic / horror / trailer / heavy orchestral. Use as a starter library for editors; expand per shoot.

> **⚠️ Verification status (v1):** All 25 tracks below are verified live on **Mixkit** (Mixkit License — free commercial use, no attribution required, [license](https://mixkit.co/license/#musicFree)). Pixabay Music, Uppbeat, and YouTube Audio Library were blocked from the automated verifier (Cloudflare / Vercel / login gates) — recommended manual additions from those libraries are listed at the bottom for an editor to fill in from a residential IP.
>
> Mixkit **does not expose per-track URLs** — the `URL` column points to the genre/tag listing page where the track appears (search by title + artist to find the play button). BPM values are genre-based estimates; confirm by ear before edit sync.
