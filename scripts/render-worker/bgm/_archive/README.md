24 tracks were fetched during the phase75 BGM library build but did
not survive review against the owner's SOP (see `docs/bgm/vibe-map.md`).
They remain here for reference (mp3 is gitignored — the files don't
enter the repo). `worker.pick_bgm()` skips this directory at runtime.

Latin, samba, beach, and cuban vibes. Rejected because the music

supports, never leads.

Jazz Brunch, …). SOP explicitly excludes jazz.
