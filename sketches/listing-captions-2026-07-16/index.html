Listing caption — design demo

4 directions for the bottom-of-frame caption on listing (home tour) videos. Real data from a live Percho listing. Click a variant to see it in a phone frame with a real listing photo.
📍 5122 Lower Creek Street · Peachtree Corners GA 30092 · $968K · 4bd/4ba/3,302 sqft · Modern
A
Editorial masthead
NYT Real Estate / Kinfolk. Serif price as hero, thin rules, section masthead. Reads like a magazine spread — high trust, calm. Bottom-only text so photo breathes.
B
Cinematic minimal (VO-ready)
Sub-30% pixel coverage. Frosted-glass pill + tiny chapter marker. Assumes voice-over will carry the info — screen gets out of the way so the photo does the emotional work.
C
Broker bold (Compass-style)
Bottom slab with strong hierarchy: giant price, address, 4-column spec row divided by hairlines. Aggressive, sales-forward. Best when the buyer is scrolling fast and you have 2 seconds to convert.
D
Chapter storytelling
Per-clip chapter caption (top-left: 'CHAPTER 01 · The Approach') pairs with photo. Bottom stays quiet with price+specs. Feels like a documentary — each clip has narrative role, primes the VO layer beautifully.
Design constraint I baked in: voice-over is coming next. So each variant is designed to leave room for VO to add information rather than repeat what's on screen. B leans hardest into this (chip only, VO carries everything). A/D leave a lot of empty photo. C is the most redundant with VO — pick C if you think VO is optional / for accessibility only.
