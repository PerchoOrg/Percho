Vicinity — Agent Profile / 4 Design Directions

Four totally different visual languages for the agent profile (/a/[slug]). Click each to view the full comp. Same mock data — Vivian Zhang, 6 Eastside listings — so you can compare apples to apples. Pick one (or mix-and-match) and I'll run accessibility audit + surface inventory + production rollout.

[ 01 / Editorial Mono ]
The Magazine
Cream paper, oversized Fraunces serif, asymmetric editorial grid. Numbered listings (No. 01–06) like a print portfolio. Closest to a New Yorker / Kinfolk / Apartamento spread.
Best when: Vivian wants to read as a craft / studio practitioner — sellers who care about how their home is told.
REF: Apartamento · Cereal · The Gentlewoman
The Photographer[ 02 / Pixieset ]
Italic Cormorant Garamond, soft cream/sand background, pure masonry of full-bleed photos with hover overlays. The literal Pixieset translation you asked for — feels like a wedding photographer's gallery.
Best when: the photos must do 90% of the talking and you want the lowest cognitive load on the buyer.
REF: Pixieset · Format · Squarespace Brine
The Brokerage[ 03 / Compass-Luxe ]
Deep navy background, Playfair Display + gold accents, structured 2-column listing cards with featured slab. Reads as institutional, ultra-credible — Compass / Sotheby's / The Agency LA territory.
Best when: you need to convince sellers Vivian plays at the $5M+ tier without saying it. Most "buyable" feeling for B-side acquisition.
REF: Compass · Sotheby's Realty · The Agency
The Studio[ 04 / Cinematic Runway ]
Pure black, Geist + Geist Mono, full-bleed cinematic listing rows alternating sides. Lime accent ticker. Reads like a film studio reel or RunwayML / Linear product page — youngest, most "tech".
Best when: Vicinity wants to position as the new-school alternative to Zillow/Compass — tech-y, video-first, brand differentiator.
REF: RunwayML · Linear · Apple Vision Pro
